### 更新(2025.9.8)馬戲團自動切換使用說明### 
- 助手請保持半自動或關閉狀態. 開始執行馬戲團後將會
- 循環切換角色(外部指定)
- 在馬戲團地圖內:
  a. 使用SET2練功裝
  b. 展開任務欗並自動領取500隻任務獎勵
  c. 如果背包倒數30格道具(假設最新領出來的)有超過10個道具是鎖定且沒有裝備的話, 腳會假設設定錯誤並嘗試把裝備穿上. 不想穿的裝備請不要鎖定
- 在非馬戲團地圖:
  a. 檢查馬戲團剩餘時間.
  b. 檢查身上是否有指定存倉道具(外面設定1個指定道具全名). 
  c. 時間用完-> 切SET3 -> 存倉
  d. 還有時間 -> 領裝 -> 進入馬戲團
  e. 存取物品會自動包含中元節憑證+活動金變道具
- 異常處理:
  a. 如果超過3分鐘沒有在馬戲團地圖內而且卡在登入畫面, 會發DC/Telegram訊息通知
- 外部預先設定 (由上至下):
  a. 主道具: 如果是金武, 請將全名打在主道具
  b. 練功裝: 設定1個道具來讓腳根據此道具是否在身上判斷是存或領裝.  請選用全部分身都會穿的, 比方說 成長符石
  c. 切換角色名單: 指定要打的角色ID (一定要同伺服器)

- 遊戲內:
  a. 每隻角色請確認SET2 是穿你要的打馬戲團裝備. SET3會將全部一件存倉的裝備脫下
  b. 確保角色有足夠的金幣領取裝備
  c. 雖然有寫SET2 防空身穿裝, 但是未測試還是請各位觀察
  d. 確認一鍵存倉裝備設定沒問題. 金武要確定放在自動存倉設定內
